# 🔐 SIEM Learning Path (TryHackMe)

## 📌 Introduction

This project shows my learning journey in **SIEM (Security Information and Event Management)** using TryHackMe.

I learned how SOC analysts use SIEM tools to monitor logs, detect threats, and investigate security incidents.
