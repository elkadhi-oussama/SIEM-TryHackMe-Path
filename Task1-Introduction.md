# Task 1 - Introduction to SIEM

## What is SIEM?

SIEM means Security Information and Event Management.

It is a tool used by SOC analysts.

It collects logs from many devices in one place.

## Why is SIEM important?

- Many devices create logs
- Logs are hard to read alone
- SIEM helps to analyze logs easily

## Key Point

SIEM helps analysts find attacks faster.
