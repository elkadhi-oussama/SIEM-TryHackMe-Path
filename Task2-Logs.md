# Task 2 - Log Sources

## Types of Logs

### 1. Host-Centric Logs
These logs come from the device.

Examples:
- User login
- File access
- Process execution

### 2. Network-Centric Logs
These logs come from the network.

Examples:
- VPN connection
- Web traffic
- SSH

## Problems Without SIEM

- Too many logs
- No central place
- Hard to analyze
