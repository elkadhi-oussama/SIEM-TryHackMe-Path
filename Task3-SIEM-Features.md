# Task 3 - Why SIEM

## What is SIEM?

SIEM means Security Information and Event Management.

It collects logs from many devices, normalizes them, and analyzes them to detect threats.

---

## Key Features of SIEM

### 1. Centralized Log Collection

SIEM collects logs from all devices in one place.

This helps analysts save time and work faster.

---

### 2. Log Normalization

Logs come in different formats.

SIEM converts them into one standard format.

This makes logs easy to read and analyze.

---

### 3. Log Correlation

SIEM connects logs from different sources.

This helps detect complex attacks.

---

### 4. Real-Time Alerting

SIEM creates alerts when suspicious activity is detected.

Analysts can quickly respond to threats.

---

### 5. Dashboards

SIEM shows data using dashboards.

This helps analysts understand the system quickly.

---

## Key Point

SIEM helps detect threats faster and improves security monitoring.
